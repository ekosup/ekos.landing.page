# Eko Supriyono - Portfolio

A landing page and portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Hero section
- About me
- Skills showcase
- Project portfolio
- Contact information
